Basics

To begin with, navigating through Carrd's dashboard is self-explanatory. There is the common tabs for: projects, notifications, help section (even a documation is available), profile settings, and an optional for an upgrade.

When starting out, if you plan to just have fun and create fanpages or sites with no intent of monetization, opting for the free access is perfectly fine. Once you plan to incorporate multiple sites, requiring branding, APIs, and advance features, it is highly recommended to upgrade.

The options for upgrade include the following:
  • Pro Lite - $9/yr
  • Pro Standard - $19/yr
  • Pro Plus - $49/yr
    • Templates & Blanks

    Carrd offers a great variety of templates for new users or for the experienced to quickly construct a page. Although there will be ones restricted to those who has upgraded, there is still good number available for free. They are categorized into the purpose of the page such as galleries, profiles, or forms just to name a few.

    Of course, you are welcome to start from scratch as well. This is great for those who wants to dive in and construct from the very bottom or if they have a set visual they want to achieve.

    It is here where I will highly recommend you to ideate what you want to accomplish. Not only is it frustrating if your work gets lost during the process, but also a slight con for many is how Carrd does not allow drafts to be created. Once you open a new file, you must be ready to publish it. You will still be able to make changes after the matter, but just be aware of this feature.
    Editing

    Once a template or blank canvas has been selected, you are ready to customize and edit however you like. The layout and features are very similar to CSS & Html for those who have tinkered with the markup languages. You are able to edit the background and add containers to which can hold features such as images, tables, text, etc. Again, the changes you make are close to making changes in CSS except here you are given graphic user interfaces as oppose to code lines. For example, you code adjust a textbox's width: --px; height: --px; border: --px; padding: --px; and margin: --px;

    There are some things only available to the Pro Version such as being able to embed actual CSS source codes as well as being able to add animations, slideshows, widgets, and form. The entire settings tab for features are locked unless you purchase the pro version, and the additional benefits include being able to add id="" & class=""

    Color options include the color wheel with multiple hues and scale for saturation. Carrd can also take color names in HEX decimals such as the following style="color: #4AADBA"

    Animations and Gifs

    Although being able to embed animations will not be available to the free version, being aware of the niches and finishing touches it can bring to the website should be considered if you are considering upgrading. The animations lessens the static feel to the site and opens a door to more variety.

    To add an animation, have the code/script of it available. You can find amazing websites that offer free, customizable animations such as this one. From there, go ahead and select an embed element. In the element is a settings tab that contains a dropdown called type. Select code type and you will see a textbox show up where the code snipet should be pasted.Publish your carrd and the addition can be viewed. Here is an example of what the script looks like: Plays when hover 

              <script
          src="https://unpkg.com/@lottiefiles/lottie-player@latest/dist/lottie-player.js">
          </script>
    
          <lottie-player src="https://assets5.lottiefiles.com/datafiles/wqxpXEEPRQf1JnQ/data.json"  background="transparent"  speed="1"  style="width: 400px; height: 300px;" hover loop  autoplay>
          </lottie-player>

    Not only are you allowed to add animated pictures, but animations on scroll are permitted as well. Select an embed element, go to style and select "hidden" and "head" in the two dropdowns respectfully, and paste the code.

    A field called Attributes is available alternative scrolling behaviors.

              data-aos=fade-right
          data-aos-easing=ease-in-out
          data-aos-duration=1300

    Gifs are easily added as well. Simply find a gif you like, download it as a JPEG, and upload the file in your image element.
